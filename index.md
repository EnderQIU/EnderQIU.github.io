## vslcpy
> A Very Simple Language Compiler Implemented by Python.


用 Python 写的一个自定义语言的编译器，支持交互式命令行。使用 ply 进行词法和语法分析，llvmlite连接后端进行代码生成，
相当于 ply 和 llvmlite 混合使用的一个简单示例。
- Github: <https://github.com/EnderQIU/vslcpy>
- version: 1.0
- Date: 2018-12-21
